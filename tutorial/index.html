上海高中生雅思考试完整入门教程

适用于准备海外本科申请的高中生及家长。

核心结论：准备海外本科申请的高中生通常考 IELTS Academic。

明确学校要求 → 做完整模考 → 找到短板 → 建立语言能力 → 学习题型策略 → 定期模考 → 报名考试
中国大陆普通雅思将在2026年9月1日起全面转为机考，因此现在开始准备的学生应直接按机考模式训练。

一、考试类型

海外本科申请一般选择 IELTS Academic。只有学校、预科、语言班或签证材料明确要求时，才选择 IELTS for UKVI。

二、考试结构

科目时间内容主要难点
Listening约30分钟4部分，40题拼写、数字、同义替换
Reading60分钟3篇文章，40题时间、定位、长难句
Writing60分钟Task 1 + Task 2逻辑、准确性、避免模板化
Speaking11–14分钟3部分展开、流利度、避免背诵

三、目标分数

目标建议规划
基础项目5.5–6.0
普通海外本科总分6.5，单项6.0
较强大学6.5–7.0
高语言要求专业7.0以上
第一阶段目标：总分6.5，单项不低于6.0。

四、先做真实诊断

  1. 使用电脑完成。
  2. 严格计时。
  3. 听力不能暂停。
  4. 阅读60分钟结束。
  5. 写作完成两篇。
  6. 口语完整录音。

没有错误分析的刷题，只是在重复犯错。

五、12周方案

第1–2周

诊断、熟悉题型、修复词汇语法拼写与发音基础。

第3–6周

按题型分科训练，保持写作和口语反馈。

第7–9周

进入限时训练，解决阅读做不完、写作时间失控等问题。

第10–12周

每周至少一次完整模考，最近三次稳定达标后再正式考试。

六、四科方法

听力

对答案后重听错题、查看文本、标注同义替换、跟读并在第二天复听。

阅读

核心是文章结构、快速定位和证据判断。True/False/Not Given 只认文章证据。

写作

Task 1 要有清晰 Overview；Task 2 推荐：Claim → Reason → Explanation → Example → Link。

口语

Part 1 回答2–3句；Part 2 使用故事结构；Part 3 使用 Opinion → Reason → Example → Qualification。

七、机考准备

  • 英文打字约25–35词/分钟。
  • 适应屏幕长文阅读。
  • 熟悉高亮、拖放和检查功能。
  • 练习戴耳机听力和边听边输入。

八、学习资料

  • 较新的剑桥雅思真题。
  • 官方样题和评分标准。
  • Anki、BBC Learning English、TED-Ed。
  • 写作与口语必须获得有效反馈。

九、每周安排

星期任务
周一听力精听
周二阅读和长难句
周三Task 2段落
周四口语录音
周五错题复习
周末限时训练、写作和口语模拟

十、常见失败原因

  1. 只背单词，不练使用。
  2. 过早刷完真题。
  3. 写作依赖模板。
  4. 口语背题库。
  5. 成绩不稳定就正式考试。

第一个星期

  1. 明确学校要求。
  2. 完成听力和阅读诊断。
  3. 完成两篇写作。
  4. 录制口语模拟。
  5. 建立错误分类表。
  6. 制定计划。
  7. 优先训练最弱两个模块。
